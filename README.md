# Susan Dynarski website

Plain static academic website for GitHub Pages.

Upload the contents of this folder to the root of `dynarski.github.io`. Do not upload the zip file itself.

Files:

- `index.html`
- `research.html`
- `writing.html`
- `teaching-resources.html`
- `about.html`
- `assets/css/styles.css`
- `assets/img/susan-dynarski-portrait.jpg`

GitHub Pages should publish automatically from the `main` branch for a repository named `dynarski.github.io`.
